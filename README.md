## Skull King Scoreboard

Lightweight single-page scoreboard for the card game Skull King. Open `index.html` directly in a browser to track bids, tricks, bonuses, and total scores across rounds.

### Development
- No build step or dependencies required.
- Edit `index.html` and refresh the page to see changes.

### Deployment
- The site is ready for static hosting (e.g., GitHub Pages) and uses only client-side code.
